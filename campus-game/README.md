# 异乡校园 · SCHULTAG

一个关于观察、记忆与书写的叙事游戏。

**在线试玩**：https://jjjjjjjjnnjnn.github.io/schultag-summer/

## 当前版本

V0.8 Reality Echo

## 核心机制

```
白天观察 → 收集素材 → 夜晚写作 → 世界记住你的观察
```

- **注意力焦点**：每天选择关注谁，影响观察成本和写作配方
- **Reality Echo**：你的选择被系统记住，改变后续叙事
- **笔记本系统**：记录你的观察、写作和人物印象

## 本地运行

```bash
cd campus-game
npm install
npm run dev
```

## 技术栈

React 19 + TypeScript + Vite + Tailwind CSS v4 + Zustand
