# 异乡校园 设计圣经

> 所有新功能必须通过本文件审核。如果一个功能不能强化核心命题，就不要做。

---

## 核心命题

> 观察会改变被观察者。
> 书写会改变书写者。

Robert 以为自己只是在记录，但他写下的每一个字都在重塑他看待世界的方式——直到虚构开始侵入现实。

这不是恋爱模拟。
这不是养成游戏。
这是关于**一个写作者如何被自己塑造**的故事。

---

## 核心循环

```
白天
├─ 观察（选择看什么）
├─ 记录（素材进入笔记本）
└─ 注意力消耗（你不能看一切）

夜晚
├─ 写作（选择素材组合）
├─ 配方匹配（素材 → 文字）
└─ Echo 触发（文字改变现实）
```

**每个环节都在做选择。每个选择都有代价。**

---

## 审美原则

三个关键词：

```
观察    记忆    写作
```

**观察**：玩家的注意力是有限资源。你选择看什么，就意味着你没看别的。

**记忆**：Notebook 是玩家的外部记忆。它不只记录信息，它记录玩家的选择偏好。

**写作**：这是转化。把观察变成文字，把文字变成现实。

## 设计语言

### 视觉

- **暗色基调**：`#0f0d0b` 主背景，温暖的深色
- **纹理**：grain overlay，notebook paper，红线 margin
- **排版**：serif 字体，宽松行高（leading-[2]），文学感
- **色彩克制**：amber 为主色调，stone 为辅，不要五彩缤纷
- **层级**：通过 opacity 和 blur 制造景深，不要平面化

### 交互

- **打字机**：对话和观察文本都有打字机效果
- **点击继续**：叙事推进是点击式的，不是自动播放
- **音效克制**：5 种合成音效，不使用外部音频文件
- **btn-press**：所有按钮有 `translateY(1px)` 微交互

### 音效

| 音效 | 用途 | 风格 |
|------|------|------|
| page | 翻页（最重要） | highpass noise 120ms |
| echo | Echo 触发 | 220Hz sine decay 300ms |
| observe | 记录素材 | sweep 320→520Hz 150ms |
| write | 写作提交 | noise + click 100ms |
| click | 按钮点击 | lowpass noise 40ms |

## 禁止事项

**绝对不做：**

```
手游化          刷数值          战斗系统
任务列表        成就弹窗        好感度系统
送礼            解锁 CG         多结局
开放世界        生存系统        技能树
自动战斗        抽卡            每日任务
```

**风格禁止：**

```
玻璃拟态        渐变按钮        卡片堆叠
侧边栏导航      仪表盘布局      默认字体
动画过度        音效过度        色彩过度
```

**这些不是"以后再说"，是永远不做。**

## 版本路线图

### V1.0 Demo（当前：V0.9.5 → V1.0）

**内容冻结。只修 Bug、文案、翻译、UX。**

目标：
- 外部试玩（5+ 玩家）
- 发布 GitHub Pages + itch.io
- 收集第一批反馈

### V1.1 Consequence Update（后果更新）

**让玩家感觉：我的文字真的改变了世界。**

升级 Echo：
- 现在：写作 → 引用一句话
- 以后：写作 → 人物行为改变 → 关系改变 → 事件改变

### V1.2 Relationship Web

**不是好感度，是人物认知网络。**

Notebook 中：
```
Maya
├─ 对 Robert：好奇
├─ 对 Ludwig：保持距离
└─ 对自己：迷茫
```

随观察和写作变化。

### V1.3 校园时间系统

**从线性章节到自由选择。**

```
周一 → 去图书馆？去食堂？去球馆？
```

产生：错过、选择、遗憾。

### V1.5 第一卷完整版

目标：15~25 小时。
不是增加 100 个系统，而是增加更多观察、更多人物、更多写作、更多后果。

### Steam

如果第一卷成功。

## 试玩观察清单

外部试玩时，观察：

```
1. 玩家在哪里停下？
   - 序章退出？
   - 第三章退出？

2. 玩家最喜欢什么？
   - Notebook？
   - Maya？
   - 写作？
   - Echo？

3. 玩家最困惑什么？
   - 焦点系统？
   - 写作配方？
   - 观察机制？
```
