## 《脸盲》章节施工设置总表



### 一、叙事方法设置



**核心叙事者**：第一人称“我”（荣加俊），有限视角，全程不离开“我”的感知范围。



**叙事策略**：“冷媒介”记录——叙述者只记录感官信息（看到的、听到的、身体感受到的），尽量不解释、不评价、不替人物做心理总结。



| 叙事层级 | 操作方法 | 对应你的风格 |

|---------|---------|------------|

| **感官记录** | 光的位置、声音的质感、物体的颜色/形状/距离 | 摄像头美学 |

| **动作记录** | 叉子放下、笔在纸上划过、椅子被往后靠、手指在桌上敲 | 用物和身体说话 |

| **对话记录** | 直接引语，不附加“他不耐烦地说”这类情绪标签 | 让声音自己呈现 |

| **内心闪念** | 只记录“我”脑子里确实闪过的念头，不展开成思考 | 比“心中一紧”更真实 |



**关键纪律**：全章不出现一次“我心想”或“我感到”。内心的东西要么通过对话漏出来，要么通过一个短促的“念头”（一句独白，不加引号的那种）闪现。

### 二、记录方法设置（分节镜头脚本）



把全章当作用文字拍一部短片。以下是五个镜头的拍摄方案。



**镜头一：课上的死水微澜（约250字）**



| 拍摄要素 | 具体内容 |

|---------|---------|

| **机位** | 固定，中等距离。从教室中后排往前看。 |

| **入画** | 讲台上老师走动、黑板上写着东西；前景是“我”和王嘉亿的课桌。 |

| **声音** | 老师讲课的声音（闷的，作为背景音）；空调的嗡嗡声；窗外极远处偶尔的脚步声。 |

| **物的焦点** | 王嘉亿手里转的那支笔；“我”面前笔记本上画的无意义线条；从窗外斜照进来的一道阳光，落在课桌角上。 |

| **动作链** | 王嘉亿凑过来小声说话→“我”回“哦。行。好。”→王嘉亿又说一句→“我”回“别说话了，老师说话了”→王嘉亿靠回去，笔还在转。 |

| **结束画面** | 王嘉亿的笔停了一下，他的目光飘向窗外。“我”低头看着笔记本上画的线条，不知道自己在画什么。 |



**镜头二：窗边的“罗生门”（约400字）**



| 拍摄要素 | 具体内容 |

|---------|---------|

| **机位** | 从“我”的眼睛出发，先近（王嘉亿的表情），再远（窗外人影），再近（收回视线）。 |

| **入画** | 王嘉亿的手指戳了戳“我”的胳膊→两人的头同时转向窗外→操场、篱笆、几个移动的人影。 |

| **声音** | 王嘉亿压低的声音：“哎哎哎，看窗外——”他的声音压得很低却有明显的兴奋；“我”的声音懒洋洋的，带着困惑；老师讲课的声音仍在继续，但被虚化。 |

| **物的焦点** | 窗外那几个“一点点大”的人影（远景）；王嘉亿指着窗外的手（近景）；“我”眯起眼睛的动作。 |

| **关键镜头** | 王嘉亿吐槽“你的脸盲真的应该治一治了”时，他的面部表情——眉毛挑起来，嘴角半笑不笑。这是老朋友才有的坦然讽刺。 |

| **结束画面** | 人影走近，消失在教学楼的入口方向。王嘉亿低声说“他们过来了”。“我”把视线收回到课本上，发现自己在同一行字上停了很久。 |



**镜头三：意外的点名（约350字）**



| 拍摄要素 | 具体内容 |

|---------|---------|

| **机位** | 从门口拉回教室内部。先拍门被敲响（全班轻微骚动），再拍老师走向门口，最后聚焦到“我”被点名时那一瞬间。 |

| **入画** | 老师在门口用德语低声交谈→身后的门外站着什么人（不写清楚，保持模糊）→老师转身，新同学被领进来→坐定→老师目光扫视全班→**目光停在“我”身上**。 |

| **声音** | 敲门声（三下，不重）→门外的德语交谈声→椅子被拉开的声音→全班的沉默（这个沉默需要被写出来）→老师的声音：“看样子Robert很期待啊。” |

| **特写** | “我”的眼睛睁了睁。“我”的手从桌上拿下来，放到膝盖上。“我”的嘴动了一下，然后说出：“O——Ok哪。” |

| **结束画面** | “我”站起来。“我”的自我介绍的英文声响起（文字用中文写），整个教室的目光都聚过来又散开。 |



**镜头四：他人的碎片与“我”的第一印象（约300字）**



| 拍摄要素 | 具体内容 |

|---------|---------|

| **机位** | “我”坐下后的视角。先看王嘉亿站起来介绍，然后看兰若瑶站起来。两个机位基本相同，但“我”对两个对象的视线停留时长和焦点不同。 |

| **入画** | 王嘉亿介绍时，身体语言更放松，手会配合说话做小幅度动作→兰若瑶介绍时，身体不太动，只是站着，手放在身前或自然垂着。 |

| **声音** | 王嘉亿的语速快、内容丰富（旅行、冰岛、烘焙）→那个德国同学打断“我也叫Meier”→老师纠正→兰若瑶的声音“我叫兰若瑶…18岁…喜欢烘焙…生态学…”语速不快，但很简洁。 |

| **“我”的内部声音** | “我”偏头小声对王说：“哇塞，这个英语口语开口就是王炸级别的。”——这句话是评价，用的是“我”的世界里的最高赞美标准：**技术水平**。 |

| **外貌描写** | 按照“摄像头清单”写：身高（目测1米5，方式是不动声色地拿她和旁边的课桌高度比一下）→长相（不像18岁，方式不是“她长得显小”，而是“她站起来的时候，旁边德国同学比她高了一大截，看着不像同一个年级”）→衣物（白色加淡蓝的冲锋衣、马尾辫）。全部用外部视角，不写“她看起来很——”。 |

| **结束画面** | “我”的视线在她身上多停了半秒。然后收回来，低头，发现桌上的笔还在原位。 |



**镜头五：引用的回响（约200字）**



| 拍摄要素 | 具体内容 |

|---------|---------|

| **机位** | “我”的课桌角度，微俯。可以看到自己的题纸，也可以看到斜前方王嘉亿和兰若瑶共看的那份题。 |

| **入画** | 王嘉亿把题目挪到两人中间→兰若瑶低头看题，“我”低头写自己的→**核心画面**：兰若瑶看了一会儿，转过脸问王嘉亿：“要不要写句式引用和句子的文章来源？”→王摇头：“不用。”→兰若瑶转回去，开始写。 |

| **声音** | 只有那一句简短的问题和回答。背景恢复到老师的声音和笔尖划过纸面的沙沙声。 |

| **“我”的内部声音** | **仅一句**：“哇塞，她居然知道在作文章分析的时候写引用！真专业！”——这是本章唯一一次“我”对兰若瑶产生正面情绪，而且它被严格限定在“专业能力”的范围内。这是“我”唯一愿意认可的评判维度。 |

| **终场画面** | “我”把视线从他们那边收回到自己的卷面上。笔继续写。窗外操场上的光，和上课前一模一样，但有一片云移开了，光线亮了一点。“我”把“引用”这个词在心里放了一下，然后接着写下一道题。 |

### 四、文章结构设置



**章节标题**：`二、脸盲`



**篇幅控制**：约1600—1800字（与你已完成的《闲聊_1》篇幅接近，保持节奏统一）。



**五节比例分配**：



| 节 | 功能 | 预计字数 | 占全章比 |

|----|------|---------|---------|

| 第一节 | 课堂氛围+王嘉亿的躁动 | 250字 | 15% |

| 第二节 | 窗边观察（全章核心戏剧） | 400字 | 24% |

| **第三节** | **被点名+自我介绍（人物亮相）** | **350字** | **21%** |

| 第四节 | 兰若瑶亮相+“我”的观察 | 300字 | 18% |

| 第五节 | “引用”事件+结尾悬停 | 200字 | 12% |

| （弹性留白）| 用于动作节奏调整 | 100-200字 | 10% |



**开头方式**：场景切入，不用过渡句。直接从“小飞飞的课”开始。



**结尾方式**：物性悬停。最后落在一个物或感官细节上——建议落在“窗外操场上的光”或“笔尖在纸上划过的声音”，与开头形成不经意的闭环。



**章节衔接**：



- **与上一章（《闲聊》）的衔接**：只有几天的时间跳跃，不需要解释。王嘉亿一开口就是在期待见面，读者自然能接上。

- **与下一章的衔接预告**：无需在本章结尾预告。下一章可以接一个三人第一次非正式互动的场景（如下课后走廊碰到、第一次一起吃饭等），完成“从观察到接触”的过渡。

### 五、创作纪律对照表（自查用）



| 标准 | 操作方式 |

|------|---------|

| **强张力** | 王嘉亿的“热”与“我”的“冷”在全章持续拉扯，直到“引用”处出现一个极微小的和解。 |

| **回味性** | “引用”事件给读者留下问题：这个女孩到底有多专业？“我”会因此改变态度吗？ |

| **讨论性** | 脸盲、误解、用技术标准衡量人——这些都可以被讨论，但小说本身不讨论。 |

| **文学性** | 五节镜头化处理，物的焦点贯穿始终，结尾悬停，比喻精准而不泛滥。 |

| **哲学性** | “脸盲”作为人际认知的隐喻；“引用”作为经验主义认识论的一个微型操作——我们通过“具体证据”认识一个人。 |

| **真实性** | 对话节奏、课堂氛围、被点名时的尴尬、心不在焉的观察——全部来自日常经验，不戏剧化。 |

| **非灌输** | 本章不替任何人下结论。兰若瑶好不好？“我”有没有改变？不回答。只呈现。 |
